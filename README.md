# UniTrust
Welcome to UniTrust — a university document verification system that makes credentials easy to verify and hard to fake.

We build tools that let universities issue digital documents, students share them, and organizations verify them instantly. The system keeps data tamper‑proof, reduces manual paperwork, and helps prevent fraud.

## What we focus on

- Multi‑level signatures (issuer + validator approvals)
- Secure document issuance and verification
- Simple integrations for universities and third parties
- Privacy‑aware sharing for students
- Transparent, auditable records
- Validator model
- Validators (authorized academic units or third‑party verifiers) confirm document legitimacy before it is marked as trusted. Each validation leaves a permanent, auditable record, creating a safer, multi‑party trust chain.

## Repository overview

- Backend: API and core services for document flow
- Contracts: Smart contracts for verifiable records
- Frontend: Web app for users and administrators
